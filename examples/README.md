# Examples

Real-world examples of AI-Framework in action.

| Project Type | Description | Key Patterns |
|--------------|-------------|--------------|
| [Web App](web-app/) | Next.js SaaS application | Full FIC workflow, multi-portal |

## How to Use

1. Pick the example closest to your project type
2. Copy the templates to your project
3. Customize for your domain
4. Follow the FIC workflow

## What's Included

Each example contains:
- **MASTER_REFERENCE.md** — Filled project knowledge base
- **TRANSFER_PACK.md** — Sample session handoff
- **sample-research.md** (web-app only) — Example research output with line numbers

## Adapting to Your Project

These examples are templates, not prescriptions. Adapt them:

| If Your Project... | Then... |
|--------------------|---------|
| Has different tech stack | Update Tech Stack section |
| Is smaller scope | Remove unused sections |
| Has more complexity | Add domain-specific sections |
| Is team-based | Add Team section |

*Start with the example closest to your use case, then customize.*
