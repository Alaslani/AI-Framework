## Key Concepts

```
The Pyramid Principle:
─────────────────────
Bad Research    = 1000s of bad lines
Bad Plan        = 100s of bad lines
Bad Code        = 1 bad line

→ Invest time at the top of the pyramid
```

```
The FIC Workflow:
─────────────────
FIND        → IMPLEMENT    → COMPOUND
(Research)    (Execute)      (Learn)
   20%           70%           10%
```
